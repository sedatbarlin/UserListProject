# UserList 📱

UserList, kullanıcı verilerini çekip listeleyen ve detaylarını gösteren bir iOS uygulamasıdır. MVVM mimarisi ve delegate pattern kullanarak yapılandırılmıştır.

## Özellikler ✨
- **Kullanıcı Listesi**: API'den kullanıcı verilerini çekerek listeleme.
- **Detay Sayfası**: Her bir kullanıcının detaylı bilgilerini gösterme.
- **MVVM Mimarisi**: Model-View-ViewModel mimarisi ile yapılandırılmıştır.
- **Delegate Pattern**: Kullanıcı seçimlerini yönetmek için delegate pattern kullanılmıştır.
- **SnapKit**: Auto Layout için SnapKit kütüphanesi kullanılmıştır.

## Teknolojiler 🛠️
- Swift
- UIKit
- SnapKit
- JSONDecoder
